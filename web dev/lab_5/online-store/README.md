# Online Store - Lab 5

## Description
This project is an online store built with Angular. It displays products from kaspi.kz with categories, like functionality, and delete options.

## Features
- 4 categories (Smartphones, Laptops, Headphones, Tablets)
- 20 products (5 per category)
- Like button with counter
- Delete button to remove products
- WhatsApp sharing
- Category switching
- Responsive design

## Technologies Used
- Angular 17+
- TypeScript
- HTML5 & CSS3
- Flexbox/Grid for layout

## Installation & Setup

### Prerequisites
- Node.js (v18 or later)
- npm (comes with Node.js)
- Angular CLI
